2. 表面层级 / Surface Elevation

深色主题通过亮度递增表达层级，而非阴影。越高层的 surface 越亮。

bg Level 0 — 页面
bg-panel Level 1 — 面板
surface Level 2 — 卡片
surface-2 Level 2 — 工具栏
surface-3 Level 3 — Hover
surface-active Level 4 — Active

7. 设计原则 / Design Principles

Token 优先
所有视觉值来自 design token。
禁止使用 Tailwind 默认色板（如 gray-800）。
使用 bg-surface 而非 bg-zinc-800。
shadcn/Radix 优先
交互组件优先使用 Radix UI 原语。
已有：Dialog, Tabs, Switch, Tooltip, ScrollArea, Sheet, Command。
需要新组件时，优先从 shadcn/ui 添加。
层级通过亮度
深色主题中，层级通过背景亮度递增表达。
bg → bg-panel → surface → surface-2 → surface-3 → surface-active
避免使用 box-shadow 表达层级。
CVA 管理变体
组件变体使用 class-variance-authority (CVA)。
配合 cn() 工具函数合并 Tailwind 类。
cn(buttonVariants({ variant }), className)
